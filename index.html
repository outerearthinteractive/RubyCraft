RubyCraft
by Outer Earth Interactive

Minecraft Server implemented in Ruby.

May not work on Windows yet! This is a reverse-engineered clean-room version of the Minecraft Server. None of the logic is read from any Minecraft code, obfuscated or unobfuscated. The only resources used to implement the protocol are freely available protocol documents on some of the Wikis.

Dependencies

Required gems: eventmachine

Install

Make sure to do this if you haven't already (running it again won't hurt anything) 'gem install eventmachine' Then just run main.rb!
