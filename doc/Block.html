    class Block

    This class provides configuration variables for the entire server. At a later date, it will be superseded with a more dynamic configuration-backend loading scheme. Distributes under the same terms as Ruby

    Variables

    • block_x = the x position of the block

    • block_y = the y position of the block

    • block_z = the z position of the block

    • block_type = the item id of the block

    • block_data = the extra data for the block

    Example

    Create a block of wood at <120,64,159>

    b = Block.new(120,64,159,5)
